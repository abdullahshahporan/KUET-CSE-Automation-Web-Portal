# KUET CSE Automation Web

A Next.js web application with Node.js backend, prepared for PostgreSQL database integration.

## Tech Stack

- Next.js (React framework)
- TypeScript
- Node.js
- PostgreSQL (to be integrated later)
- Tailwind CSS

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
